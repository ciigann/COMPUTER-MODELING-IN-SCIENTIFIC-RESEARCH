### 2.1	Формулировка дискретной задачи

r– радиус, mass – масса, alpha – угол, v_0 – начальная скорость, y_0 –начальная высота, dt2– шаг сетки (параметр дискретизации в методе Эйлера), dt1 – период расчетов, tk – конечное время для расчетов, di – относительная погрешность по энергии, E – полная энергия, V – вектор скорости, r – радиус-вектор, Fd – сопротивление воздуха, m – масса, h – высота, g = 9,81 м/с²

Используя Численное решение дифференциальных уравнений – метод Эйлера, в котором функции непрерывного аргумента заменяются функциями целочисленного аргумента – сеточными функциями, запишем уравнения дискретной модели для времени, компонент вектора скорости, координат, радиус вектора и энергии. Задав dt2– шаг сетки (параметр дискретизации в методе Эйлера), dt1 – период расчетов, tk – конечное время для расчетов и начальные координаты, сведем систему дифференциальных уравнений к системе линейных уравнений:

## 3	ПРОГРАММИРОВАНИЕ ЗАДАЧИ

Программа для визуализации математической модели движения тела, брошенного под углом к горизонту написана на языке python и скомпилирована для удобства в файл расширением .exe . Для использования программы пройдите по пути и запустите файл dist\model\model.exe. 

<img width="624" height="397" alt="image" src="https://github.com/user-attachments/assets/b49815bb-1c87-4e4e-a9ad-6be740314bea" />

Установите начальные условия модели и нажмите кнопку «отправить» для построения траектории полета объекта. Под данной кнопкой программа выведет точку пересечения графика с осью Х, среднюю полную энергию и среднеквадратическую относительную погрешность. 

<img width="624" height="396" alt="image" src="https://github.com/user-attachments/assets/76cddf65-550a-4e5c-bf6a-9a57cfab3f59" />

## 4	РАСЧЕТЫ ПО ПРОГРАММЕ И ОЦЕНКА ТОЧНОСТИ РАСЧЕТОВ

Проведем эксперимент, построим 3 модели с одинаковыми параметрами и с разным шагом сетки: 0.1, 0.01 и 0.00001. Создадим таблицу для оценки точности расчетов программы. Посмотрим среднеквадратичную относительную погрешность и закон сохранения энергии. Графики показывают траекторию модели с заданным шагом и с шагом на порядок меньше. Можно заметить, что при уменьшении шага сетки среднеквадратическая относительная погрешность становится меньше. Модель точнее описывает траекторию полета и точку приземления. 

<img width="587" height="575" alt="image" src="https://github.com/user-attachments/assets/c4cf5a4e-c45e-4fa6-9010-30ea426d2cc3" />

Выполним оценку точности расчетов координат и скорости объекта. Заметим, что с шагом сетки 0.1 Vx сразу выдает десятичную точность, остальные компоненты радиус-вектора и вектора скорости выдают нулевую точность. При шаге сетки 0.01 программа выдает одинарную точность для компоненты Vy. При шаге сетки 0.00001 программа выдает одинарную точность для компонент: X, Y, и двойную точность для компоненты Vy. Можно заметить, что при уменьшении шага, точность модели возрастает.
